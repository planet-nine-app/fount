Key Features:

  • Selection Spells: Add items to in-memory magistack collection
  • Selection + Navigation: Selection spells can also navigate to next card with bdoPubKey
  • Magicard Navigation: Navigate between cards using BDO pubKeys
  • MAGIC Protocol: Integration with The Advancement extension
  • Auto-Detection: Automatically finds and enhances spell elements
  • Visual Feedback: Hover effects and click animations
